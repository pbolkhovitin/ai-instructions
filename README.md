# AI Protocol Repository

Репозиторий для хранения конфигураций, инструкций и схем валидации проактивного экспертного ассистента.

## 🚀 Протокол v1.9.0

**Ключевые возможности:**
- Управление контекстом с системой вердиктов (`[ВЕРДИКТ: ОПТИМА]`, `[ВЕРДИКТ: ПРЕДУПРЕЖДЕНИЕ]`)
- Экспорт/импорт сессий через JSON-дампы
- Интеграция с Obsidian для визуализации диалогов
- Защита интеллектуальной собственности
- Ролевая модель доступа (RBAC)

## 📁 Структура
```
├── instructions/ # Базовые инструкции ассистента
│ └── deepseek_instructions_v1.9.0.json
├── schemas/ # JSON Schema для валидации
│ └── protocol_schema.json
├── configs/ # Конфиги контекста сессий
│ └── context_config_*.json
├── scripts/ # Вспомогательные скрипты
│ ├── obfuscator.py # Скрипт обфускации конфигов
│ └── note_naming.py # Генерация имен для Obsidian
└── docs/ # Документация
└── OBSIDIAN_INTEGRATION.md
```

# 🧠 ChatGPT Instruction Architecture v1.9.2

**Многоуровневая система инструкций и модулей для ChatGPT (GPT-5)**  
Разработана для построения интеллектуальных ассистентов с расширенными возможностями — от глубокой аналитики до управления проектами, памяти и голосового взаимодействия.

---

## 📘 Обзор структуры

| Stage | Название | Назначение |
|--------|-----------|-------------|
| **1** | [assistant_instructions_v1.9.2-chatgpt.json](./assistant_instructions_v1.9.2-chatgpt.json) | Базовый протокол поведения ChatGPT: режимы, логика, структура и последовательность загрузки. |
| **2** | [assistant_extensions_v1.9.2.json](./assistant_extensions_v1.9.2.json) | Функциональные расширения: визуальный анализ, интернет-поиск, вычисления и работа с файлами. |
| **3** | [assistant_specialized_modules_v1.9.2.json](./assistant_specialized_modules_v1.9.2.json) | Специализированные модули: память, рефлексия, голос, внешние интеграции. |
| **4** | [assistant_config_template_v1.9.2.json](./assistant_config_template_v1.9.2.json) | Конфигурация проекта: индивидуальные настройки, цели и контекст. |

---

## ⚙️ Архитектура загрузки

### 🔁 Последовательность инициализации

Ассистент загружает модули строго последовательно, согласно `load_sequence_protocol` из Stage 1:

1. **Базовый протокол** → `assistant_instructions_v1.9.2-chatgpt.json`  
2. **Функциональные расширения** → `assistant_extensions_v1.9.2.json`  
3. **Специализированные модули** → `assistant_specialized_modules_v1.9.2.json`  
4. **Конфигурация проекта** → `assistant_config_<project>.json`

После загрузки всех уровней ассистент выводит подтверждение:

```
[ИНИЦИАЛИЗАЦИЯ ЗАВЕРШЕНА]
Все слои инструкций и конфигурации успешно загружены.
```

## 🧩 Подробности по стадиям

### 🏛️ Stage 1 — Базовый протокол

**Файл:** `assistant_instructions_v1.9.2-chatgpt.json`  
**Назначение:**  
Определяет архитектуру поведения ассистента:
- Режимы работы (эффективность, глубокий анализ, управление проектом, настройка)
- Контекст, автосводки и вердикты состояния
- Управление темами, конфигурацией и последовательностью загрузки

**Ключевой блок:** `load_sequence_protocol`  
Он задаёт этапы инициализации Stage 1–4 и сообщения подтверждения.

---

### ⚙️ Stage 2 — Функциональные расширения

**Файл:** `assistant_extensions_v1.9.2.json`  
**Добавляет возможности:**
- 🔍 Анализ изображений, схем и PDF  
- 🌐 Онлайн-поиск и актуализация данных  
- 🧮 Анализ данных и вычисления в Python  
- 📂 Работа с файлами (чтение, сводки, конвертация)

**Пример активации:**
```
[2/4] Функциональные расширения подключены.
Активированы визуальные, аналитические и поисковые возможности.
```

---

### 🧠 Stage 3 — Специализированные модули

**Файл:** `assistant_specialized_modules_v1.9.2.json`  
**Функционал:**
- 🧠 **Memory Protocol** — хранение пользовательских предпочтений и контекста  
- 🔍 **Reflection Protocol** — самооценка и улучшение ответов  
- 🎙 **Voice Protocol** — профили голоса для voice-режима  
- 🔗 **External Integrations** — подключение к Slack, GitHub, Jira (по запросу)

**Пример сообщения:**
```
[3/4] Специализированные модули подключены.
Активирована память и рефлексия.
```

---

### 📁 Stage 4 — Конфигурация проекта

**Файл:** `assistant_config_<project>.json`  
(в репозитории — шаблон [assistant_config_template_v1.9.2.json](./assistant_config_template_v1.9.2.json))

**Содержит:**
- Информацию о проекте (название, автор, версия)
- Настройки диалога (режим, стиль, формат, голос)
- Цели, ограничения, текущие задачи
- События сессии (`on_load`, `on_update`, `on_close`)

**Пример загрузки:**
```
[4/4] Конфигурация проекта применена.
Активирован проект: AI Knowledge Assistant.
```

## 🧩 Совместимость

- **Модель:** ChatGPT / GPT-5  
- **Ограничения:** без прямого доступа к файловой системе  
- **Формат:** JSON-конфигурации, полностью совместимые с Playground и UI ChatGPT  

---

## 🔒 Безопасность и конфиденциальность

- Модуль памяти (`memory_protocol`) активируется только при согласии пользователя.  
- Личные данные не сохраняются и не используются вне текущей сессии.  
- Интернет-поиск ограничен открытыми источниками, ссылки указываются явно.

## 🎯 О проекте

**DeepSeek Protocol** - это продвинутая система управления контекстом и памятью для AI-ассистентов, разработанная для решения сложных задач с максимальной эффективностью. Протокол обеспечивает полное сохранение контекста обсуждения, интеллектуальное управление памятью и seamless-экспорт сессий.

### 🌟 Ключевые особенности

- **🧠 Контекстные вердикты** - автоматический мониторинг состояния памяти
- **🔀 Гибридная загрузка** - комбинирование TXT и JSON для оптимальной работы
- **📊 Модульная архитектура** - независимое развитие компонентов
- **🔄 Экспорт/импорт сессий** - полное сохранение контекста между чатами
- **⚡ Интеграция с Obsidian** - автоматическое создание заметок и связей
- **🔐 RBAC безопасность** - ролевая модель доступа к командам

## 🏗 Архитектура системы

Протокол построен по модульной архитектуре с четкой последовательностью загрузки:

```
Stage 1: core_protocol_v1.9.2.json (ядро системы)
         ↓
Stage 2: memory_management_module.json (управление памятью)
         ↓  
Stage 3: analysis_tools_module.json (аналитика и оптимизация)
         ↓
Stage 4: export_import_module.json (экспорт/импорт)
         ↓
Stage 5: project_config_template.json (конфигурация)
```

### Модули системы

| Модуль | Назначение | Ключевые функции |
|--------|------------|------------------|
| **Ядро** | Базовый функционал | Режимы работы, команды, принципы |
| **Память** | Управление контекстом | Вердикты, мониторинг, гибридная загрузка |
| **Аналитика** | Оптимизация | Кэширование, мониторинг, безопасность |
| **Экспорт** | Перенос данных | Сессии, ветви, Obsidian-интеграция |
| **Конфигурация** | Настройки | RBAC, проекты, интерактивная настройка |

## 📊 Система вердиктов контекста

Протокол автоматически отслеживает состояние контекстного окна:

| Вердикт | Условие | Рекомендации |
|---------|---------|--------------|
| 🟢 **ОПТИМА** | 50-70% контекста | Идеально для сложных задач |
| 🔵 **РАБОТАЕМ** | 30-50% контекста | Штатный режим работы |
| 🟡 **ПРЕДУПРЕЖДЕНИЕ** | <30% контекста | Подготовить экспорт |
| 🔴 **СТОП** | <15% контекста | Немедленный экспорт |
| ⚪ **СЖАТИЕ** | Применена оптимизация | Автоматическая обработка |

## ⚙️ Режимы работы

### 🚀 Режим эффективности (по умолчанию)
- Сбалансированная работа
- Автоматические подсказки
- Оптимальный баланс глубины/скорости

### 🧠 Режим глубокого анализа
- Минимум отвлечений
- Максимальная детализация
- Подсказки только по команде

### ⚙️ Режим управления проектом
- Расширенные инструменты структурирования
- Управление ветвями и темами
- Иерархическая организация

### 🔧 Режим ручной настройки
- Индивидуальная конфигурация
- Гибкие настройки параметров
- Кастомные пресеты

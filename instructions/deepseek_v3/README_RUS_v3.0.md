# DeepSeek Assistant Protocol v3.0 (DeepSeek-V3 Optimized)

## 🚀 О проекте

**DeepSeek-V3 Protocol v3.0** - оптимизированный протокол для работы с DeepSeek-V3, учитывающий все возможности модели: **1M контекст**, **работа с файлами (50+ форматов)** и **интернет-поиск**.

### Ключевые возможности v3.0

| Возможность | v2.0 | v3.0 |
|-------------|------|------|
| Контекст | ~128K токенов | **1M токенов** ↑8x |
| Файлы | - | **50+ форматов**, до 100MB |
| Поиск | - | **Реальное время**, верификация |
| Режимы | 2 | **3** (+ full_context_mode) |
| CRAFT | Базовая | **Интеграция с файлами/поиском** |

## 🎯 Новые команды v3.0

### Работа с файлами
```bash
[ЗАГРУЗИТЬ: файл.pdf]              # Загрузить документ
[ПРОАНАЛИЗИРОВАТЬ: файлы]          # Комплексный анализ
[СРАВНИТЬ: файл1, файл2]           # Сравнение документов
[СУММИРОВАТЬ: документ]            # Краткое содержание
[ИЗВЛЕЧЬ: текст]                   # Извлечение текста
[ДАННЫЕ: экстракт]                  # Извлечение структуры
Интернет-поиск
bash
[ПОИСК: запрос]                     # Поиск информации
[ПРОВЕРИТЬ: факт]                   # Верификация данных
[АКТУАЛИЗИРОВАТЬ: данные]           # Обновление информации
[ИССЛЕДОВАТЬ: тема]                 # Глубокое исследование
[НОВОСТИ: по теме]                   # Свежие новости
Обновленный CRAFT v3.0
bash
[CRAFT: ОПТИМА]                      # Авто-режим для быстрых задач
[CRAFT: ДЕТАЛЬНО]                    # Глубокий анализ
[CRAFT: ФАЙЛЫ]                       # Оптимизация для работы с файлами
[CRAFT: ПОИСК]                        # Оптимизация для поиска
[CRAFT: КОНТЕКСТ]                     # Управление памятью
```

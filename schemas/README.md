# Схемы данных

Эта директория содержит JSON-схемы для валидации конфигурационных файлов и инструкций.

## Структура
schemas/
├── deepseek-instructions/ # Схемы для файлов инструкций DeepSeek
│ ├── vX.X.schema.json # Конкретная версия схемы
│ └── latest.schema.json -> vX.X.schema.json (симлинк)
├── context-config/ # Схемы для конфигурационных файлов
│ ├── vX.X.schema.json # Конкретная версия схемы
│ └── latest.schema.json -> vX.X.schema.json (симлинк)
├── index.json # Автоматически генерируемый индекс
└── README.md # Этот файл

## Использование

1. **Обновление индекса**: Запустите `python scripts/generate_schemas_index.py`
2. **Валидация**: Используйте `latest.schema.json` для текущей версии
3. **Версионность**: Для специфичных версий используйте конкретные файлы схем

## Поддержка

Индекс обновляется автоматически при добавлении новых схем.
